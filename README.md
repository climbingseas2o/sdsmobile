About the SDS Mobile3 To-Do App

My Android to-do list app for the course. Built using Kotlin and XML. 
 
How to run:
·      Download the project
·      Open in Android Studio
·      Run on emulator (or Android device)
 
Functionalities:
·      “Add Item” button to add a new to-do item
	o   Write your task title and tap Save
	o   In case error, press Cancel
·      To do item added on list
·      Tap the checkbox to mark a task as done
